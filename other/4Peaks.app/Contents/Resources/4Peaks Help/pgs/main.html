4Peaks Main Window & Preferences

Figure 1 shows the main window of 4Peaks. The actions button is located in the lower left corner, more to the right the slider for the zoom function. The quality, speech and sequence information buttons can be found in the lower right corner. The search function is positioned in the upper right corner.



Figure 1. 4Peaks main window


Action Button

In the actions menu you will find options to edit your sequence data. In 4Peaks you can edit , insert, and delete bases. You are also able to crop your sequence to the region you are interested in, or to the region which is still readable.

Because sequence reactions can be performed with sense and anti-sense primers, the results will also be generated in a sense and a reverse complement

Due to the differences in amino acid condon usage between species, you can set your preferred translation dictionary, amino acids are appointed according to the species you are interested in. The frame your sequence is translated in can be set with the 'Set Frame' button just besides the speech button. See figure 3.

You can BLAST your sequence against one of the selectable genomes, do a MegaBLAST or BLAST 2 sequences against eachother with the help of the BLAST option.

Remarks:

  1. Most functions found in the menu when you push the button can also be accessed via shortcut keys and via the edit menu.

  2. Sequence files opened in ABI, ALF, CTF and BIO format cannot be saved as such files again after editing.



Figure 2. The Action button menu


Zoom Function

Some bases in sequence files are positioned very close to each other. 4Peaks has a zoom option so you are able to stretch or crop you sequences to have a better overview. The maximum zoom factor for zoom in and zoom out is 2. Horizontal zoom is the default option, but you can also zoom verticalliy by pressing the Shift key as you slide the zoom slider.


Quality Function

Some file types (ABI/SCF) support quality data. The quality data will be shown along with the sequence. Preferences for the quality function can be set in the preferences section under the 'traces' tab. The Quality data button alows you to show/hide the quality data and to set a treshold for your data. In the latter case, a window will pop-up and let you set the preferred treshold with the slider on the left. You can view the data again, bad quality will get a red mask; good quality will remain white, or you can crop the file to the position you think still contains good quality data.



Figure 3. Set quality data


Speech Function

Because BLASTing a sequence can be a time consuming process, sometimes you just want to compare a small part of your obtained sequence with a template sequence. 4Peaks has the opportunity to let your sequence file be spoken, so you can read the template file along on your computer or on a paper hardcopy. To start the speech function, select the base you want to start with and press spacebar or the speech button of the main window. Preferences for the speech function can be set in the preferences section.


Info Drawer

The drawer can be opened by clicking on one of the three drawer buttons: sequence; translation; information. The drawer will slide out of the main window and text format sequence and translation are immediately ready. The translation frame can be set using the 'Set Frame' button, see figure 4.



Figure 4. Set translation frame


Search function

In 4Peaks you can search within your sequence and within the translation of the sequence. To search in either of one, select the right option within the search oval (see figure 5) and start typing the sequence you are looking for. If the sequence is present in the sequence file it will be selected automatically in the main window, but also in the drawer when it is open.



Figure 5. Search feature


Preferences

4Peaks gives you the opportunity to set some preferred options in five categories:

  • General. For automated version check and your preferred BLAST setting.

  • Data. For conversions (wrinting to ABI, ALF, CTF and BIO formats is not supported), scaling and background substraction.

  • Traces. For base colors and selction options.

  • Translation. Amino acid usage, 1 or 3 letter code, and default translation dictionary.

  • BLAST. Set your preferred BLAST type and BLAST server.

  • Speech. Select the speed and voice for the speech option. Optional: use (editable) words in stead of letters only. Words should start with either A, T, C, or G!

  • Motifs. Set your default search type and manage & create your search motifs.

  • Plugins. Manage your plugins. For more information see Plugins.



Figure 6. Preference options
