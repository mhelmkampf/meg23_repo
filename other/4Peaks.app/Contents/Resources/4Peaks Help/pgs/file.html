4Peaks File Menu

Opening files

4Peaks is capable to open all DNA sequencing file formats that are generally used, including:

  • ABI
  • SCF
  • ALF
  • PLN
  • EXP
  • ZTR
  • ZTR1
  • ZTR2
  • ZTR3
  • BIO
  • TXT

Remark: Several sequence programs for the Mac require transformation of files generated on a Windows computer to Mac files. 4Peaks does not require this, because 4Peaks can handle both file formats.


Saving files

4Peaks can save your (edited) sequence files in all common sequence file formats, except ABI, ALF, CTF and BIO. So once you start editing a file of one of these formats you can only save them in an other file format.


Exporting files

Files can be exported as text files. You can select between saving the sequence or the translation.


Toggle Drawer

Via the file menu and the shortcut keys ⌘+D, you can toggle the drawer of 4Peaks. The drawer gives extra information about your sequence data. You can choose between the DNA sequence, the protein translation and information about the sequence file and the sequence run itself. For more information about the drawer, please have a look at the Help information about the Main Window.


Page Setup & Printing

Printing of your files is very easy in 4Peaks. You can also Print > Save as PDF to create a high quality PDF you can use for publication.
